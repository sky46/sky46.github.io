Project 2: Wood Joinery Frame

The second project was a wood joinery frame. It's a little frame made out of wood that demonstrates a few different joints commonly used in woodworking. The frame itself doesn't really serve any practical purpose; it's more of a project where we learn and practice creating these joints, so we know how they work, and how to create and use them in future projects. We also learned about the various advantages and disadvantages of each.

Unlike the BBQ scraper, where we all had our own designs, for this project everyone used the same design and dimensions.
